## Installation
{:.mt-lg-0}

### Dependencies

Aside from mockzilla and mockzilla-webextension itself, you'll only need [jest](https://jestjs.io/).

### NPM

Install via npm:

```
npm i -D mockzilla mockzilla-webextension
```

### Getting Started

To get started, follow the links in the features list on the right.
